## Requirement

- [python 3.6+](https://www.python.org/downloads/)

## Installation

1. `git clone https://github.com/GreatWizard/django-tourist-taxes.git`
1. `python3 -m venv django-tourist-taxes`
1. `cd django-tourst-taxes`
1. `pip install -r requirements.txt`
